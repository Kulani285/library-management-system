# Library Management System

A full-stack Library Management System built end-to-end following the Software Development Life Cycle (SDLC) and Software Testing Life Cycle (STLC), with every phase documented as it happens.

Author: Kulani Chauke

## Planned Tech Stack
- Frontend: React
- Backend: Node.js / Express
- Database: PostgreSQL
- Payments: Stripe (test/sandbox mode)
- AI features: LLM API (Anthropic Claude API or OpenAI API) for natural-language search, AI-generated summaries, and a member chatbot
- Email: Transactional email provider (e.g., SendGrid/Postmark)

## Project Methodology
This project is being built in public, one SDLC/STLC phase at a time, with each phase's artifacts committed to the `docs/` folder before development begins. The goal is to show the full process a real software project goes through, not just the finished code.

## Core Features (planned)
- Catalog management with search and availability tracking
- Borrowing/returning with due dates and fine calculation
- Real payment gateway integration for fine payments
- AI-powered natural-language search, AI-generated book summaries, and a member chatbot assistant
- Email notifications (borrow confirmations, due-date reminders, overdue notices, payment confirmations)
- Admin reporting (overdue books, most-borrowed books)
